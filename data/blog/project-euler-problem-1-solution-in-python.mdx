## This problem states

```text
"If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9.
The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below 1000."
```


## My solution

I thought that it would be best to use a list comprehension to make a list of all the numbers that are evenly
divisible by 3 and 5, and then use a for loop to add them all together. I have included the code below with
comments

```python
#This creates a list of all the numbers between 1 and 1000 exclusive if they
#are evenly divisible by 3 or 5
numbersDivBy3And5 = [i for i in range(1, 1000) if i % 3 == 0 or i % 5 == 0]

#printed the list just to be sure that it worked out
print(numbersDivBy3And5)

#This section is my for loop that will sum the numbers in the list.
#I initialized the variable first and then ran a for loop to go
#through each number and add it to the variable I am using as
#the final sum
numbersDivBy3And5Sum = 0
for numbers in numbersDivBy3And5:
    numbersDivBy3And5Sum += numbers


# I then printed the final answer
print(numbersDivBy3And5Sum)
```
